Line Breaks & Paragraphs

Important: Line breaks in markdown vary by platform:

  • For a single line break: End a line with two spaces, then press Enter/Return
  • For a new paragraph: Leave a blank line between text (press Enter/Return twice)

CRLF vs LF: Different platforms handle line endings differently:

  • Windows uses CRLF (Carriage Return + Line Feed)
  • Mac/Linux use LF (Line Feed only)
  • This rarely affects markdown rendering, but can matter when sharing files across platforms
